## Deskripsi
Tugas ini bertujuan untuk memodifikasi program sederhana C++ "Hello World" agar dapat menerima input nama dari pengguna, kemudian menampilkan sapaan menggunakan nama yang telah di input tersebut tersebut.

## Perubahan pada Kode
Perubahan yang dilakukan dari program awal:
- Menambahkan library #include <string>
- Mendeklarasikan variabel bertipe string dengan nama nama
- Menggunakan getline(cin, nama) untuk menerima input nama lengkap (termasuk spasi)
- Mengubah output program menjadi Hello <nama>

## Hasil
Program berhasil menerima input dari pengguna.
